# Automated Quality Inspection System

## Project Description

This project uses OpenCV and Python to inspect parts moving on a conveyor belt and identify defects automatically.

## Features

- Live camera inspection
- Grayscale conversion
- Gaussian blur
- Thresholding
- Contour detection
- Bounding boxes
- Pass/Fail logic
- Real-time monitoring

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

Press Q to quit.
